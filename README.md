# 🌍 TravelMate Chatbot

## 📌 Project Overview

TravelMate Chatbot is a simple travel assistance chatbot developed using HTML, CSS, and JavaScript. It helps users get basic travel recommendations and tourist information through an interactive chat interface.

## ✨ Features

* Interactive chatbot interface
* Travel destination suggestions
* Responsive and user-friendly design
* Real-time chat responses
* Easy to use

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.
4. Start chatting with TravelMate.
